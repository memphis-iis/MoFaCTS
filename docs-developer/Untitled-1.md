# Remaining AI Content Creator Prompt Follow-Ups

No unresolved items remain from the first prompt-quality finding.

Resolved decisions:

- Typed answers should usually be one canonical answer, not a list of variants.
- Prefer short answers unless the lesson is specifically teaching the longer form. For example, use `ATP` unless the goal is to learn `adenosine triphosphate`.
- Foreign-language content should preserve required accents and orthography because those are part of the language.
- Multiple-choice questions should be independent items. Do not create repetitive questions, duplicate prompts, or answer choices that differ only by trivial formatting.
- Do not add vague "answer normalization" prompt rules. If MoFaCTS needs synonym, accent, case, spelling, or punctuation tolerance, that should be handled deliberately by the existing answer assessment logic or by a separate explicit content field, not by unclear generator wording.
