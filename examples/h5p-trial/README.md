# H5P Trial

Copyable template for adapting an H5P interaction into a MoFaCTS trial result.

Current status: typed template covered by the repository typecheck through the migration bridge.

Start with `H5PTrialAdapter.ts`. It converts an external H5P score event into an explicit MoFaCTS-like trial result without touching app persistence.

Use this when a contributor wants to convert external H5P scoring events into explicit trial results.
