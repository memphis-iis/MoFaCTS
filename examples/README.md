# Examples

Target home for copyable contributor examples.

Current status: typed migration examples. The examples are intentionally small, but they are real `.ts` files and should remain covered by typecheck/lint so they do not drift into stale documentation.

Belongs here:

- Minimal unit type examples.
- Minimal trial type examples.
- Model policy examples.
- H5P trial examples.
- Content adapter examples.

Does not belong here:

- Production app code.
- Test fixtures that are only for automated test suites.

Examples are part of the contributor-facing architecture, not extra decoration.
