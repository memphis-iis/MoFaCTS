# Architecture Docs

Target home for architecture documentation that explains the shape of the codebase.

Belongs here:

- Directory structure documentation.
- Runtime boundary documentation.
- Dependency direction and source-root migration notes.
- Architecture decision records when a decision affects multiple areas.

Does not belong here:

- Contributor tutorials. Put those under `docs/contributors/`.
- Deployment runbooks. Put those under `docs/deployment/`.
- Public API examples. Put those near `examples/` or the relevant package.
