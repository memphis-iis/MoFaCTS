# Code of Conduct

## Our Commitment

We are committed to fostering an open and welcoming community for all contributors and users.

## Expected Behavior

- Be respectful and professional.
- Assume good intent and discuss ideas constructively.
- Give and receive feedback in good faith.
- Focus on evidence and reproducibility in technical discussions.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks, intimidation, or doxxing.
- Deliberate disruption of community processes.

## Scope

This code of conduct applies in project spaces, including issues, pull requests, discussions, and other official communication channels.

## Reporting

For conduct concerns, open a private report via repository maintainers. For security issues, use the process in `SECURITY.md`.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues, and contributions that violate this code of conduct.
